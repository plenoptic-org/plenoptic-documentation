Portilla-Simoncelli Texture Model

The following notebooks introduce the Portilla-Simoncelli texture model, as first described in Portilla and Simoncelli, 2000, and its implementation in plenoptic.

If you are unfamiliar with the model, I recommended reading the first two notebooks and then reading the notebooks that catch your interest.

If you are familiar with the model and its matlab implementation, start with Using the Portilla-Simoncelli model in plenoptic and check out Notable differences between Matlab and Plenoptic Implementations to see how to use the model in plenoptic and the differences between the two implementations.

If you are planning on synthesizing Portilla-Simoncelli texture metamers for your own research, please read Portilla-Simoncelli optimization details and Portilla-Simoncelli Model Limitations.
